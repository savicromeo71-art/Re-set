Statische Landingpage fuer GitHub Pages.

## Deployment ueber GitHub Pages

1. Neues GitHub-Repository erstellen.
2. Diese Dateien in das Repository hochladen:
   - `index.html`
   - `styles.css`
   - `logo.svg`
   - `am-stickpack.png`
   - `pm-stickpack.png`
   - `concept-mockup.png`
   - `founders.jpg`
   - `.nojekyll`
3. In GitHub zu `Settings` -> `Pages` gehen.
4. Unter `Build and deployment` bei `Source` `Deploy from a branch` waehlen.
5. Branch `main` und Ordner `/ (root)` waehlen.
6. Speichern.

Danach ist die Seite oeffentlich unter deiner GitHub-Pages-URL erreichbar.
